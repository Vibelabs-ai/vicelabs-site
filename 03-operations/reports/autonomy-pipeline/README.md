# Autonomy Pipeline Reports

**Source of truth:** `03-operations/reports/autonomy-pipeline/index.json`

This folder stores the synthesized autonomy notes that sit above the daily lessons-learned loop.

## Contents

- One markdown note per day: `YYYY-MM-DD.md`
- One bundle input per day: `inputs/YYYY-MM-DD.bundle.json`
- One index file for navigation and dashboard integration: `index.json`
- Notes should stay short, opinionated, and validation-driven
- Concrete tracker follow-ups belong in the action tracker unless they are durable process changes
